
プログラム動作させるには

OpenAIのAPIを動かすための鍵を取得して`.env`に記載する必要があります
HuggingfaceのAPIを動かすための鍵を取得して`.env`に記載する必要があります

``` .env
OPENAI_API_KEY=""
OPENAI_API_MODEL="gpt-3.5-turbo"
OPENAI_API_TEMPERATURE=0.5

HUGGINGFACEHUB_API_TOKEN=""
```
